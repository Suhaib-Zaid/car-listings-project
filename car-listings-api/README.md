A simple backend API built with Node.js, Express, and SQLite for managing car listings.

- Node.js
- Express.js
- SQLite

## How to Run

```bash
npm install
node index.js
```

Test it with Postman or any HTTP client at:
```
http://localhost:3000/cars
```
